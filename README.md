# Sons of Zion

Landing Page fuer die Sons of Zion Seminare — ein Werk von MANTD.

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- GSAP (Animationen via CDN)
- Google Fonts: Cormorant Garamond, IBM Plex Mono

## Projektstruktur

```
index.html              Hauptseite
assets/images/          Bilder (Logo, Portraits)
docs/                   Projekt-Dokumentation
```

## Lokal starten

Die Seite direkt im Browser oeffnen oder einen lokalen Server starten:

```bash
npx serve .
```
